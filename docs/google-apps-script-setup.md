# Google Apps Script連携手順

このサイトは静的HTMLなので、メール送信とスプレッドシート保存はGoogle Apps ScriptのWebアプリで行います。

## 1. スプレッドシートを作成

1. Googleドライブで新しいスプレッドシートを作成します。
2. ファイル名を例として「TOACH 無料業務診断 受付」にします。
3. メニューから「拡張機能」→「Apps Script」を開きます。

## 2. Apps Scriptを貼り付け

1. Apps Scriptの `コード.gs` を開きます。
2. このリポジトリの `google-apps-script/Code.gs` の内容をすべて貼り付けます。
3. 管理者にも通知したい場合は、先頭の `ADMIN_EMAIL` に通知先メールアドレスを入れます。
4. 保存します。

## 3. Webアプリとしてデプロイ

1. Apps Script画面右上の「デプロイ」→「新しいデプロイ」を押します。
2. 種類の選択で「ウェブアプリ」を選びます。
3. 実行ユーザーは「自分」を選びます。
4. アクセスできるユーザーは「全員」を選びます。
5. 「デプロイ」を押します。
6. 初回は権限承認が出るので、スプレッドシート操作とメール送信を許可します。
7. 発行された「ウェブアプリURL」をコピーします。

## 4. フロント側にURLを設定

`script.js` の次の行に、コピーしたURLを入れます。

```js
const GOOGLE_APPS_SCRIPT_WEB_APP_URL = "https://script.google.com/macros/s/xxxxxxxx/exec";
```

## 5. 動作確認

1. サイトで診断を最後まで回答します。
2. 問い合わせフォームに名前、メールアドレス、相談内容を入力します。
3. 「無料業務診断を申し込む」を押します。
4. スプレッドシートに1行追加されることを確認します。
5. 入力したメールアドレスに自動返信が届くことを確認します。

## 保存される内容

- 送信日時
- お名前
- メールアドレス
- 相談内容
- 従業員規模
- 推定従業員数
- 年間損失額
- 診断結果
- 主な原因
- 損失額の内訳
- 設問回答
- ページURL

## 注意

フロント側の `fetch` は `no-cors` で送信します。ブラウザからレスポンス本文は読めませんが、Apps Script側ではスプレッドシート保存と自動返信メール送信が実行されます。
